{\selectlanguage{english}
The goal of this project was to create a desktop application to help manage a driving
school (OSK). The app was built in C\# using WPF and the MVVM pattern, with data stored
in a local SQL~Server database accessed through Entity~Framework~Core.

The application has three user roles. The administrator can add, edit and delete students,
instructors and vehicles, and can manage all driving reservations. The instructor can view
their weekly schedule and cancel their own driving sessions. The student can independently
book available driving slots using an interactive calendar.

The project includes data validation mechanisms, secure password storage and
data integrity protection in the database.
}
